JS: Multiple Stopwatches

In the last few years, I have been an active member of a small YouTube singing community. Recently, I created this application to help myself and others with scripting songs (deciding who sings which lines). When I script a song, I place a high priority on each singer getting approximately the same amount of time as every other singer. Before, I had to play through the song once for every singer so that I could time how long they had to sing. With this tool, by allowing me to create multiple stopwatches and assign a keyboard shortcut to each one, I can quickly figure out how much time each singer has with only one playthrough of the song. (Theoretically. My coordination is only good enough to handle around 3-4 singers at once - which still saves quite a bit of time!)

Bash: Script to assist me with grading

This small script was my first foray into bash scripting. I wrote it to help me grade assignments as a TA. The school website allows me to download all student submissions at once; this gives me one folder with a subfolder for each student. Each subfolder contains all of the files that the student submitted, with the filename that the student submitted with. The script that I wrote takes in a CSV file containing each student's first and last names. For each student listed in the CSV file, it pulls their submission files from their subfolder and copies them to a specified location, renames them into a more reasonable (at least, for me) format. The renaming puts the row number from the CSV file at the beginning of the file name, so that ordering by file name puts all the students in the same order as they are in the CSV file. The whole thing is very specific to my usecase, but it saves me many clicks and searches to find the students and submission files that I am supposed to grade.
